Real-Time BSL Captioning from a Webcam

This project explores whether British Sign Language (BSL) can be transcribed into English captions in real time using a standard webcam and machine learning.

Current Capabilities

  • Webcam-based hand detection
  • Hand landmark extraction using MediaPipe
  • Sequence classification using TensorFlow
  • Real-time caption output
