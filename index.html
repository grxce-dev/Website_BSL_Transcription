Abstract

It is estimated that there are around 25,000 people in the UK who primarily communicate using British Sign Language (BSL), and yet mainstream video conferencing tools provide no real-time transcription for signers, creating a significant accessibility gap in social, educational, and professional interactions. This project investigates the feasibility of real-time BSL transcription into English captions using a standard webcam and real-world conditions. The goal is to explore whether computer vision and machine learning techniques, such as convolutional neural networks, can recognise hand shapes, movements, and body language to accurately translate into real-time captions. While prior research has achieved promising results for American Sign Language, BSL is underrepresented and differs substantially in grammar, vocabulary, and structure. This project aims to address that gap by focusing specifically on BSL and using the available datasets to train and evaluate models. By investigating the technical challenges and limitations, including dataset size and availability, variability in signing including body language, and real-time processing, the project contributes to understanding how inclusive communication technologies can be developed.

Current Capabilities

  • Webcam video capture (OpenCV)
  • Hand landmark extraction (MediaPipe)
  • Feature vector generation (.npy)
  • Initial TensorFlow/Keras classifier

Future Developments

  • Expanded BSL vocabulary
  • Sentence-level recognition
  • Real-time captioning overlay
  • Mobile device support
